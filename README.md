# Journal

Journal is a simple php based personal diary or log book.

### Key Features

* Self hosted
* Entrys are 'private'
* language support (de and en)

### Requirements

* Webspace with some space (1 - 2 MB)
* PHP 8
* MySQL Database

#### Upcoming Features

* Password reset
* Language support (fr and it)
* Add User Accounts
* Add Location to posts
* Add Markdown
